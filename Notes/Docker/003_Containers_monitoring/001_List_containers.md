# Lista kontenerów 
Wyświetlenie listy wszystkich uruchomionych kontenerów wraz z parametrami:   
* CONTAINER ID,     
* IMAGE,       
* COMMAND,  
* CREATED,  
* STATUS,  
* PORTS  
* NAMES  
```commandline 
docker container ls  
``` 
Flaga **-a** pozwala na wyświetlenie wszystkich kontenerów, włącznie z nieaktywnymi.  
```commandline 
docker container ls -a  
```
